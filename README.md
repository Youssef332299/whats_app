# whats_app
This application is a modern real-time messaging platform inspired by apps like WhatsApp, designed to provide fast, secure, and reliable communication between users.

The app is built using Flutter with the BLoC state management pattern, ensuring scalable architecture, clean code, and high performance.

Key Features

• Real-Time Messaging – Send and receive instant text messages with a smooth chat experience.
• Voice & Media Sharing – Share images, messages with friends.
• Voice & Video Calls – Make high-quality internet calls anytime and anywhere.
• Group Chats – Create group conversations and stay connected with multiple users simultaneously.
• Secure Communication – Messages and calls are protected using end-to-end encryption so only participants can access the content.
• Online Status & Read Receipts – See when contacts are online and when messages are delivered or read.
• Modern UI with Animations – Smooth animations and responsive design provide a dynamic user experience.
• BLoC Architecture – Implemented using the BLoC pattern for clean separation between business logic and UI, making the app scalable and maintainable.

The goal of this project is to demonstrate how a full-featured messaging application can be developed using Flutter and BLoC, combining modern design, real-time communication, and secure messaging into a powerful mobile experience.
